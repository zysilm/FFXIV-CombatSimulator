# Spec: UI Layer

## Overview
All UI is rendered via ImGui through Dalamud's UiBuilder. The plugin
provides a main configuration window, a combat HUD overlay, and a
combat log window.

### 2. Enemy HP Bar Overlay

Rendered at the NPC's world position (converted to screen coordinates).

```
                    Striking Dummy
                  ████████████░░░░░░
                    45,230 / 100,000
```

Implementation:
```csharp
void DrawEnemyHpBar(SimulatedNpc npc)
{
    // Convert NPC world position to screen coordinates
    var worldPos = npc.Position + new Vector3(0, npc.Height + 0.5f, 0);
    if (!gameGui.WorldToScreen(worldPos, out var screenPos))
        return; // Off-screen

    // Draw HP bar
    var hpPercent = (float)npc.State.CurrentHp / npc.State.MaxHp;
    var barWidth = 200f;
    var barHeight = 16f;
    var barPos = screenPos - new Vector2(barWidth / 2, 0);

    // Background
    drawList.AddRectFilled(barPos, barPos + new Vector2(barWidth, barHeight),
        ImGui.ColorConvertFloat4ToU32(new Vector4(0.2f, 0.2f, 0.2f, 0.8f)));

    // HP fill (green → yellow → red)
    var color = hpPercent > 0.5f
        ? new Vector4(0, 0.8f, 0, 1)
        : hpPercent > 0.25f
            ? new Vector4(0.8f, 0.8f, 0, 1)
            : new Vector4(0.8f, 0, 0, 1);
    drawList.AddRectFilled(barPos,
        barPos + new Vector2(barWidth * hpPercent, barHeight),
        ImGui.ColorConvertFloat4ToU32(color));

    // Border
    drawList.AddRect(barPos, barPos + new Vector2(barWidth, barHeight),
        ImGui.ColorConvertFloat4ToU32(new Vector4(1, 1, 1, 0.5f)));

    // Name above
    var nameSize = ImGui.CalcTextSize(npc.Name);
    drawList.AddText(screenPos - new Vector2(nameSize.X / 2, nameSize.Y + 4),
        0xFFFFFFFF, npc.Name);

    // HP text centered
    var hpText = $"{npc.State.CurrentHp:N0} / {npc.State.MaxHp:N0}";
    var hpSize = ImGui.CalcTextSize(hpText);
    drawList.AddText(barPos + new Vector2((barWidth - hpSize.X) / 2, 1),
        0xFFFFFFFF, hpText);
}
```

## ImGui Window Registration

```csharp
public class CombatSimulatorPlugin
{
    private void InitializeUI()
    {
        pluginInterface.UiBuilder.Draw += OnDraw;
        pluginInterface.UiBuilder.OpenConfigUi += OnOpenConfig;
    }

    private void OnDraw()
    {
        if (config.ShowMainWindow)
            mainWindow.Draw();

        if (combatEngine.IsActive)
        {
            if (config.ShowEnemyHpBar)
                hpBarOverlay.Draw();

            if (config.ShowCombatLog)
                combatLogWindow.Draw();

            playerHpOverlay.Draw();
        }
    }
}
```

## Input Handling

- All UI is mouse-clickable (standard ImGui interaction)
- No custom keybinds required for MVP
- Future: configurable hotkey to toggle simulation on/off

## Performance Considerations

- HP bar overlay: only draw for NPCs within 50 yalms
- Combat log: cap at 1000 entries, auto-scroll to bottom
- WorldToScreen calls are cheap (single matrix multiply)
- Minimize per-frame allocations (reuse StringBuilder, etc.)
