# Spec: UI Layer

## Overview
All UI is rendered via ImGui through Dalamud's UiBuilder. The plugin
provides a main configuration window, a combat HUD overlay, and a
combat log window.

## Windows

### Window Sections (CollapsingHeaders)

1. **Status** — Always visible top section
   - Active/Inactive indicator with color
   - DPS and combat duration (when active)
   - Help text when inactive

2. **Simulation Controls** — DefaultOpen, positioned at top
   - Start/Stop Combat + Reset All buttons
   - Damage Multiplier slider (0.1–5.0)
   - Critical Hits + Direct Hits checkboxes
   - Player simulated HP bar (when active)

3. **Active Targets** — DefaultOpen
   - List of registered combat NPCs
   - Per-NPC: HP progress bar (green/yellow/red) + level + name + AI state
   - Cast bar (when NPC is casting)
   - Model Override sub-tree (ModelCharaId input, Apply/Restore buttons)

4. **Target Behaviors** — Under Active Targets
   - "Move Targets Near Player" checkbox + distance slider
   - "Aggro Propagation" checkbox + range slider
   - Help text explaining each behavior

5. **Settings** — Collapsed by default
   - Show Enemy HP Bar checkbox
   - Show Player HP Bar checkbox
   - Show HUD Player HP Bar checkbox
   - Show Combat Log checkbox
   - Show Shortcuts Bar checkbox
   - Show Death Cam Toolbar checkbox
   - Default NPC Level slider (1–100)
   - Default HP Multiplier slider (0.1–10.0)
   - Default NPC Behavior combo (Dummy/Melee/Ranged/Boss)

6. **Animation Commands** — Collapsed by default
   - Melee/Ranged Attack command inputs
   - Player Death command + Death Emote ID
   - Player Victory / Target Victory commands
   - Explanatory help text for each field

7. **Glamourer Integration** — Collapsed by default
   - Enable/disable death preset + design selection
   - Enable/disable reset (respawn) preset + design selection
   - Refresh Designs button
   - Status indicators

### 2. Enemy HP Bar Overlay

Rendered at the NPC's world position (converted to screen coordinates)
via ImGui background draw list.

```
                    Striking Dummy
                  ████████████░░░░░░
                    45,230 / 100,000
```

- HP color: green (>50%) → yellow (>25%) → red
- Only draws for NPCs within render distance
- Toggle via config: ShowEnemyHpBar

### 3. Player Simulated HP Overlay

Shows player's simulated HP above the character (separate from real HP).

- Toggle via config: ShowPlayerHpBar
- Optional HUD version (fixed screen position): ShowHudPlayerHpBar

### 4. Combat Log Window

Scrollable log of combat events.

```
┌─ Combat Log ──────────────────────────────────────────┐
│ [12:34:56] You use Fast Blade → Striking Dummy        │
│            523 damage                                  │
│ [12:34:58] Striking Dummy auto-attacks → You           │
│            145 damage                                  │
│ [12:35:03] Striking Dummy is defeated!                 │
│                                                [Clear] │
└────────────────────────────────────────────────────────┘
```

- Auto-scroll to bottom
- Clear button
- Capped at 1000 entries

## ImGui Window Registration

```csharp
// In Plugin constructor
pluginInterface.UiBuilder.Draw += OnDraw;
pluginInterface.UiBuilder.OpenMainUi += OnOpenMainUi;
pluginInterface.UiBuilder.OpenConfigUi += OnOpenConfig;

// OnDraw
if (config.ShowMainWindow) mainWindow.Draw();
if (combatEngine.IsActive)
{
    if (showHpBars) hpBarOverlay.Draw();
    if (config.ShowCombatLog) combatLogWindow.Draw();
}
```

### 5. Shortcuts Bar

Optional floating compact window with quick-access combat buttons.
Toggled via "Show Shortcuts Bar" in Settings.

```
┌─ Shortcuts ─────────────────┐
│ [Start] [Reset All] [Reboot]│
└─────────────────────────────┘
```

- **Start/Stop**: Toggle simulation on/off
- **Reset All**: Reset all combat state (HP, cooldowns)
- **Reboot**: Stop + Start in one click (useful for quick restarts)
- Window is movable and rendered as a separate ImGui window
- Toggle via `config.ShowShortcuts`

### 6. Death Cam Preset Toolbar

Optional floating toolbar for quickly switching between death cam presets
using left/right arrow buttons. Toggled via "Show Death Cam Toolbar" in Settings.

```
┌─ Death Cam Presets ──────────┐
│ [<]   Cinematic Side   [>]   │
└──────────────────────────────┘
```

- **< / >**: Cycle through saved presets, loading each on click
- Centered preset name label between arrows
- Arrows are disabled at list boundaries
- Shows "No presets saved." if none exist
- Toggle via `config.ShowDeathCamToolbar`

### Death Cam Preset List

The preset list in the Death Cam settings section uses `ImGui.BeginListBox` with
`Selectable` items instead of a `Combo` dropdown. This provides a scrollable list
showing up to 8 items, with a scrollbar when more presets are saved.

## Performance Considerations

- HP bar overlay: only draw for NPCs within range
- Combat log: cap at 1000 entries, auto-scroll
- WorldToScreen calls are cheap (single matrix multiply)
- Minimize per-frame allocations
